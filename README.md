# MNTN

This is a front-end project built with React and Vite. The goal is to create a modern, responsive landing page for a fictional outdoor equipment brand.

## Project Structure

```
eslint.config.js
index.html
package.json
README.md
vite.config.js
public/
  fonts/
  images/
src/
  App.css
  App.jsx
  index.css
  main.jsx
  assets/
  sections/
```

- **public/**: Contains images and fonts used in the project.
- **src/**: Main source code, divided into components and sections.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Available Scripts
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the production version.
- `npm run preview`: Previews the production build locally.

## Technologies Used
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) (presumed by utility classes)

## Component Structure
- **App.jsx**: Main application component.
- **sections/Header.jsx**: Page header with images and gradients.
- **sections/Content.jsx**: Main landing page content.

## Fonts and Images
Fonts and images are located in the `public/fonts` and `public/images` folders.
